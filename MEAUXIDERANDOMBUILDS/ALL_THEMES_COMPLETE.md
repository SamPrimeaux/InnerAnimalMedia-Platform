## üöÄ Usage

### Apply Theme
```javascript
// Set theme via data attribute
document.body.setAttribute('data-theme', 'inner-animal-dark');
document.documentElement.setAttribute('data-theme', 'inner-animal-dark');
```

### CSS Variables
All themes use CSS custom properties:
- `--bg-surface` - Background surface color
- `--bg-panel` - Panel/card background
- `--color-primary` - Primary brand color
- `--color-secondary` - Secondary accent color
- `--color-accent` - Accent color
- `--color-text` - Main text color
- `--color-muted` - Muted/secondary text
- `--color-border` - Border color
- `--radius` - Border radius
- `--shadow-clay` - Clay shadow (for clay themes)
- `--glass-blur` - Glass blur amount (for glass themes)
