Getting Started

1. Authentication

All API requests require authentication. Use OAuth 2.0 or API keys.

curl https://inneranimalmedia.com/api/auth/token \ -H "Content-Type: application/json" \ -d '{"email": "user@example.com", "password": "password"}'

2. Making Requests

Include your authentication token in the Authorization header:

curl https://inneranimalmedia.com/api/tools \ -H "Authorization: Bearer YOUR_TOKEN_HERE"

3. Response Format

All responses are JSON formatted with a consistent structure:

{ "success": true, "data": { ... }, "pagination": { ... } }
